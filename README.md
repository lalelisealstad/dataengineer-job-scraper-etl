# dataengineer-skills-scraper-etl

Run first time:
$ python3 -m venv .venv
$ source .venv/bin/activate 
$ pip install -r requirements.txt
$ pip install ipykernel

run after installation:
$ source .venv/bin/activate
$ python "app.py"
